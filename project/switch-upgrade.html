Wing-Wide Network Infrastructure Upgrade

SCOPE: Physical Layer, VLAN Config, Enterprise Switching

1. The Mission

I participated in a critical infrastructure refresh for the Wing, replacing aging switch hardware to support increased data throughput and improve network reliability for mission-critical operations.

Figure 1: Network Rack Implementation


2. Implementation Steps

  • Physical Installation: Racking and stacking new enterprise switches, managing cable runs for airflow and organization.
  • Configuration: Port security implementation and VLAN tagging to segregate voice, data, and management traffic.
  • Validation: Conducting post-installation connectivity tests to ensure zero downtime for critical assets.

3. Operational Impact

This upgrade reduced network latency and eliminated packet loss issues associated with the legacy hardware. It also standardized the configuration across the Wing, making future troubleshooting significantly faster.



Result: Enhanced operational readiness and secured the physical layer of the base network.
